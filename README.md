# Portofolio Pendaftaran CPNS

cara menjalankan portofolio:
- clone github ini
- gunakan sql yang sudah tersedia pada folder PDM
- gunakan localhost untuk mengakses website

fitur yang tersedia
- registrasi sebagai peserta
- pendaftaran peserta
- cetak bukti pendaftara (pdf)
- login sebagai admin
- verifikasi peserta
